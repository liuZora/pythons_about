# Notes for numpy and pandas

#### 介绍
所有文件来源于github的numpy和pandas库的jupyter学习笔记[http://github.com/npbcts/PythonDataAnalysisLearnNote](http://github.com/npbcts/PythonDataAnalysisLearnNote)，本人导出了html和.py格式放gitee以便读者阅览、查询、跑代码。请勿用于商业用途，如侵权则删。

#### 使用说明

1.  在html和.py文件，有中文名对应的函数/功能，方便查阅。
2.  建议用jupyter软件打开jupyter文件，便于展示代码、结果和解读。
3.  python数据分析学习资料建议阅读《利用Python进行数据分析（原书第3版）》by-Wes McKinney
